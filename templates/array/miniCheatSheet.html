Two Pointer Technique

Idea: Use two pointers from both ends for efficiency.

  • Check palindrome arrays.
  • Find pairs with a given sum (sorted array).
  • Separate even & odd numbers.

Works best when array is sorted.

Prefix & Suffix Arrays

Idea: Precompute sums/products to answer queries efficiently.

  • Product of array except self.
  • Range sum queries.
  • Handle divisions carefully.

Precompute prefix/suffix for O(n) solution.

Sliding Window

Idea: Maintain a window of size k and slide it.

  • Maximum sum subarray of size k.
  • Count subarrays meeting criteria.
  • Find min/max efficiently.

Update sums incrementally.

Next Greater / Smaller Element

Idea: Find next element greater/smaller using stack.

  • O(n) using stack.
  • Useful in histogram/span problems.

Merge Intervals

Idea: Merge overlapping intervals efficiently.

  • Sort by start time.
  • Merge sequentially.

Useful in scheduling problems.

Subarray Sum Patterns

Idea: Count subarrays with a target sum.

  • Prefix sum + hashmap.
  • Count subarrays with sum = k.
  • Handle negatives carefully.

Works for variable size subarrays.

Rotate / Reverse Array

Idea: Rotate left/right or reverse subarrays.

  • 3-step reversal trick.
  • Useful in cyclic shift problems.

Avoid shifting one by one ‚Äî use O(n) reversal.

Dutch National Flag / 3-Way Partition

Idea: Sort arrays with 3 unique values in one pass.

  • Use low, mid, high pointers.
  • Swap to correct positions.
  • One-pass, in-place algorithm.

Used in QuickSort 3-way partitioning.
